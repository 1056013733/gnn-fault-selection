# Reproducibility Notes

This repository keeps code-facing names generic and uses `SEGR` for the public
ranking method. Some generated result directories retain internal run
identifiers because those names are part of the experiment provenance.

The ranking code constructs scores from runtime-visible graph and structural
features. The cache structural signal is derived from circuit-local structural
rank geometry and excludes name-length artifacts rather than using fixed
per-feature coefficients. Random-vector fault counts are used for offline
evaluation and for supervised LOCO baseline labels, not for constructing the
SEGR ranking.

The large node-level Fig. 4 long table is generated by
`scripts/build_rank_diagnostics.py` and is intentionally excluded from version
control. The committed Fig. 4/5 materials include the summary tables, compact
per-circuit diagnostics, point summaries, and rendered PNGs.
